# month-closing / _subflow-3-planning

## Cél

Következő havi célok kitűzése. Realisztikus, mérhető.

## Lépések

1. Az előző subflow tanulságait vegyük figyelembe (mi ment, mi nem)
2. **3-5 fő cél** a következő hónapra, mindegyik:
   - SMART (specifikus, mérhető, elérhető, releváns, határidős)
   - Hozzárendelt P-szint
   - Hozzárendelt domain (`tasks`, `wallet`, `health`, ...)
3. **Várható nagyobb események** a hónapban (calendar-ből + user input)
4. **Kockázatok / blokkolók** előrelátható módon

## Approval gate

A célokat mutasd meg a usernek **döntésre**, mielőtt commitolunk.
