# domains/

Life-area definíciók. Minden domain leírja:

- mit fed le
- milyen formátumban tárolja az adatokat (`data/{domain}.md`)
- milyen flow-k érintik
- milyen mezők kötelezőek

A domain-ek az **organizer** projekt route-jait tükrözik, hogy később az adatok
egyszerűen migrálhatók legyenek.

## Domain-ek

| Domain | Leírás | Adatfájl |
|---|---|---|
| [tasks](tasks.md) | Feladatok, prioritás, deadline | `data/tasks.md` |
| [calendar](calendar.md) | Naptár események, ismétlődők | `data/calendar.md` |
| [notes](notes.md) | Jegyzetek, note-book-ok | `data/notes.md` |
| [diary](diary.md) | Napló bejegyzések | `data/diary.md` |
| [shopping](shopping.md) | Bevásárló-listák | `data/shopping.md` |
| [stock](stock.md) | Otthoni készlet | `data/stock.md` |
| [wallet](wallet.md) | Pénzügy, bevétel/kiadás | `data/wallet.md` |
| [wishlist](wishlist.md) | Kívánságlisták | `data/wishlist.md` |
