# Shopping lists — index

Bolt/csatorna szerint szétválasztott bevásárló-listák. Az alapelv:
`current/principles/shopping-lists.md`.

| Fájl | Bolt / csatorna | Mit fed le | Ütem |
|---|---|---|---|
| `tesco.md` | Tesco (online/bolt) | élelmiszer + alapanyag + drogéria | 2-3 hetente |
| `clothing.md` | ruhabolt / cipőbolt | ruházat (zokni, alsógatya, póló, kabát, pulóver, cipő) | ad-hoc |
| `ikea.md` | IKEA | háztartási eszközök, bútor-kiegészítő (kuka, tányér, evőeszköz) | esetenkénti |

Új lista hozzáadása:
1. Új `.md` fájl ide a megfelelő struktúrával
2. Bejegyzés ebbe a táblába
3. Ha rendszeres ütem van, sor a `current/principles/recurring-tasks.md` táblájába
4. Ha rendszeres stock-tételei vannak, `current/stock/items.md`-ben jelölni a `preferredStore`-t
